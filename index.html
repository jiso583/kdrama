🟢 Conectar con Supabase

Supabase almacena tus datos y fotos en la nube de manera segura. La sincronización entre tablet y PC es automática y en tiempo real. El plan gratuito incluye 500MB de almacenamiento para tus imágenes.

1

Ve a supabase.com → New Project → elige nombre y región.

2

En el panel izquierdo → SQL Editor → pega y ejecuta el SQL de configuración que aparece abajo.

3

Ve a Settings → API → copia la Project URL y la anon/public key.

4

Pega los valores aquí y toca Conectar. ¡Listo en todos tus dispositivos!

📋 Ver SQL de configuración (copiar y ejecutar en Supabase) 
-- Tabla de dramas
create table if not exists dramas (
  id text primary key,
  es text, en text, kr text,
  year int, tipo text,
  eps_total int, eps_seen int,
  rating int, plats text[],
  notes text, images text[],
  season int, seasons jsonb,
  completed boolean default false,
  created_at bigint, updated_at bigint
);

-- Tabla de actores
create table if not exists actors (
  id text primary key,
  name text, name_kr text,
  birth text, nac text, notes text,
  photo text,
  drama_photos jsonb,
  created_at bigint, updated_at bigint
);

-- Activar realtime
alter publication supabase_realtime add table dramas;
alter publication supabase_realtime add table actors;

-- Políticas de acceso público (para uso personal)
alter table dramas enable row level security;
alter table actors enable row level security;
create policy "allow_all_dramas" on dramas for all using (true) with check (true);
create policy "allow_all_actors" on actors for all using (true) with check (true);

⬇️ Descargar la App

Descarga el archivo de la app a tu PC o tablet para instalarlo como aplicación. Guárdalo en una carpeta fija y ábrelo siempre con Chrome o Edge.

📋 Después de descargar:

1

Haz clic derecho sobre el archivo descargado → Abrir con → Google Chrome

2

Busca el ícono ⊕ en la barra de direcciones → toca "Instalar"

3

¡Listo! Aparecerá el ícono 🌸 en tu escritorio como app independiente

📱 Instalar como App (PWA)

Si ya abriste la app en Chrome, puedes instalarla directamente desde aquí sin necesidad de descargar.
