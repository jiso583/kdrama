🟢 Conectar con Supabase

Supabase almacena tus datos y fotos en la nube de manera segura. La sincronización entre tablet y PC es automática y en tiempo real. El plan gratuito incluye 500MB de almacenamiento para tus imágenes.

1

Ve a supabase.com → New Project → elige nombre y región.

2

En el panel izquierdo → SQL Editor → pega y ejecuta el SQL de configuración que aparece abajo.

3

Ve a Settings → API → copia la Project URL y la anon/public key.

4

Pega los valores aquí y toca Conectar. ¡Listo en todos tus dispositivos!

⬇️ Descargar la App

        
Descarga el archivo de la app a tu PC o tablet para instalarlo como aplicación. Guárdalo en una carpeta fija y ábrelo siempre con Chrome o Edge.

        

          
        

        

          
📋 Después de descargar:

          

            
1
Haz clic derecho sobre el archivo descargado → Abrir con → Google Chrome

            
2
Busca el ícono ⊕ en la barra de direcciones → toca "Instalar"

            
3
¡Listo! Aparecerá el ícono 🌸 en tu escritorio como app independiente

          

        

      


      

        
📱 Instalar como App (PWA)
